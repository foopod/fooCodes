Description

Jono Shields, an independent Kiwi developer that creates games in his spare time.

History

Early history

Since I got my first laptop I loved playing games. I quickly became hooked on an FPS called Urban Terror, which started as a quake 3 mod, but quickly grew steam and became its own game. It wasn't long until I was making maps for the game and creating with others in the community. This was my first taste of being able to be part of creating something interactive, just being able to show people and see how they reacted was an amazing experience.

Making Homage

Normally when I am working on a project I make a prototype, something that accomplishes what I initially intended to do. But then I hit a wall, at this point it requires the same or more effort to get the game to a point where it is polished enough to be happy to put it infront of people. Homage is the game that broke this wall, it is the game that pushed me to spend hours removed edge case bugs that were barely noticeable. Homage is dedicated to the games I played growing up, more specifically to a particular game that was my mums, on a handheld from the 80s. The first game I ever remember playing, the simple but challenging mechanics had be playing for hours. I chose to make Homage because I wanted to release something, so I chose a simple mechanic and stuck to it. I needed something simple because I was going to put it on the play store the day I started it. Homage was an 8 hour solo gamejam. The game has grown a lot since then, the mechanics have evolved and progression has been added. This was necessary to meed the expectations of modern mobile gamers that are easily distracted if they don't feel their time is being well utilized. But Homage will always at its core be a classic.

What's to come?

The next few months will largely be dedicated to Homage, adding some of the smaller features that I feel it lacks, cleaning up some of the code and fixing any bugs that arise. I want to have a play with some procedural generation and maybe something open world. I guess we will see where the future takes me.
